## 작성법
- 폴더와 파일명은 영문으로 작성.
- Github-Flavoured Markdown으로 작성.(확장자는 .md)
- 짧고 간결한 핵심적인 문장을 사용. 
- 필요한 설명이 있으면 관련된 정보가 포함된 외부 링크를 사용.
- 최소 99% 이해한것만 업로드(잊혀지지않기를..).
